# RajBhog 🍱

RajBhog is a modern full-stack grocery & food ordering platform built using Spring Boot, React, MySQL, JWT Authentication, Razorpay, Wallet System, Coupons, Returns & Refund Management.

---

# 🚀 Features

## 👤 Authentication & Authorization

- JWT Authentication
- Role-Based Access Control
- User & Admin Modules

## 🛒 Shopping Features

- Product Browsing
- Category Filtering
- Search Functionality
- Shopping Cart
- Checkout Flow

## 💳 Payment System

- Razorpay Integration
- Wallet Payment
- COD Support
- Payment Tracking

## 🎁 Offers & Coupons

- Coupon System
- Discount Calculation
- Banner Coupons

## 📦 Order Management

- Order Tracking
- Dynamic Order Timeline
- Order Cancellation
- Return Request System
- Return Approval/Rejection

## 💰 Wallet System

- Wallet Refunds
- Wallet Transactions
- Wallet Balance Tracking

## 📄 Invoice System

- PDF Invoice Generation
- Invoice Download

## 📱 UI/UX

- Fully Responsive Design
- Modern UI
- Mobile Friendly

# 📁 Project Structure

```bash
RajBhog/
 ├── rajbhog-backend/
 └── rajbhog-frontend/
```

---

# ⚙️ Backend Setup

```bash
cd rajbhog-backend
mvn spring-boot:run
```

Backend runs on:

```bash
http://localhost:8080
```

---

# ⚛️ Frontend Setup

```bash
cd rajbhog-frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

# 🌍 Deployment

- Backend → Render
- Frontend → Netlify
